C Programming Language

In this chapter we will learn about C programming language. So,

What is C programming language actually?

C programming is a general purpose programming language, supporting structural programming and recursion.

In 1972 Dennis Ritchie first introduced C programming language to the world.

Yeah,you can say it's too old.Despite being too old C is highly efficient and cross Platform supported.
For this reason, it is widely used in application development and for fields like system programming, socket programming, C is the first choice of programmers.




























C++ Programming Language

In this chapter we will learn about C++ programming Language. So,

What is C++ programming language actually?

C++ programming is a general purpose programming language, it has imperative, object-oriented and generic programming features, while also providing facilities for low memory manipulation.

In 1979 C++ programming language first come with name "C with classes" because it was the object oriented version of C.
In 1983 "C with classes" renamed with "C++"(++ being the increment operator of C).
In 1985, the first version of C++ Programming language was released.
Creator of C++ language is "Bjarne Stroustrup"

It was desined with bias toward system programming and embedded, resource-constrainted and large systems, with performance, efficiency and flexibility of use as it's design highlights.
In simpler words, C++ was created for making System Programming easier.
